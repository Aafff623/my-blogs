# knowledge/ — 跨项目知识沉淀

记录 AI Coding 环境下学到的**可迁移**设计经验，供本人复习、供其他项目 Agent 快速对齐。

| 文件 | 内容 |
|---|---|
| [`ai-coding-asset-design.md`](ai-coding-asset-design.md) | 资产目录与流程设计的精炼总结（主文档） |

## 维护规则

1. **仅用户主动提出**时新增或大幅改写 knowledge 下文件。
2. Agent 会话中学到的东西：**先输出草稿供 Review**，用户确认后才写入。
3. 项目内 operational 规范变更 → 改 `AGENTS.md` / `workflow.md`；**可迁移的方法论** → 同步摘要到 knowledge（经 Review）。
4. 不在 knowledge 重复 `workflow.md` 全文；用链接 + 原则级总结。
