# Triage 标签

技能内部使用五种 canonical 的 triage 角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 我们 tracker 中的标签 | 含义                       |
| -------------------------- | -------------------- | -------------------------- |
| `needs-triage`             | `needs-triage`       | 需要维护者评估             |
| `needs-info`               | `needs-info`         | 等待提问者补充信息         |
| `ready-for-agent`          | `ready-for-agent`    | 已描述清楚，AI 可直接接手  |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现               |
| `wontfix`                  | `wontfix`            | 不处理                     |

当 skill 提到某个角色（例如“应用 AFK-ready 的 triage 标签”）时，使用上表中对应的标签字符串。

如果你实际使用的词汇不同，请编辑右侧这一列。
